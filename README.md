# Maltese Garden Defense

A browser-playable 2D Maltese lane-defense prototype.

## Play

Open `index.html` in a browser, or use the GitHub Pages link once published.

## Local Dev

This is a plain HTML/CSS/JavaScript game. No install step is required.

Useful playtest URL flags:

- `?combo=frost-shatter`
- `?festa=early`
- `?shadow=early`

They can be combined, for example:

```text
?combo=frost-shatter&festa=early&shadow=early
```

## Notes

The game is inspired by broad lane-defense mechanics, with original Maltese-themed units, enemies, art direction, names, and balance.
